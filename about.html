How to Add New Posts

To add a new post to this blog:

  1. Create a new HTML file in the posts directory (e.g., post4.html)
  2. Add the post metadata to data/posts.json
  3. Follow the format of the existing post files

Each post should include:

  • A unique ID
  • Title
  • Publication date
  • Excerpt (short description)
  • Image URL
  • Content (HTML)
